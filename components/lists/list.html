List

List are powerful component to display series of content



Default List

Add list class to ul tag and list-items to li tag to use this component

  • List Title
  • List Item 1
  • List Item 2
  • List Item 3
  • List Item 4
                    
                        <ul class="list">
                            <li class="list-title list-item">List Title</li>
                            <li class="list-item">List Item 1</li>
                            <li class="list-item">List Item 2</li>
                            <li class="list-item">List Item 3</li>
                            <li class="list-item">List Item 4</li>
                        </ul>

Horizontal List

Use horizontal-list class to ul tag to use this component

  • List Item 1
  • List Item 2
  • List Item 3
  • List Item 4
                    
                        <ul class="horizontal-list">
                            <li class="list-item">List Item 1</li>
                            <li class="list-item">List Item 2</li>
                            <li class="list-item">List Item 3</li>
                            <li class="list-item">List Item 4</li>
                        </ul>

Spaced List

Spaced list contains space between list items.Use space-list class with list class in ul element to use this component

  • List Title
  • List Item 1
  • List Item 2
  • List Item 3
  • List Item 4
                    
                        <ul class="list spaced-list">
                            <li class="list-title list-item">List Title</li>
                            <li class="list-item">List Item 1</li>
                            <li class="list-item">List Item 2</li>
                            <li class="list-item">List Item 3</li>
                            <li class="list-item">List Item 4</li>
                        </ul>

Stacked List

Stacked List can be used for notification messages. Use stacked-list class in ul tag to use this component

  • This is a success alert message
  • This is a warning alert message
  • This is an info alert message
  • This is a danger alert messge
                    
                        <ul class="stacked-list">
                            <li class="list-item">
                                <div class="alert alert-success-fl">This is a success alert message</div>
                            </li>
                            <li class="list-item">
                                <div class="alert alert-warning-fl">This is a warning alert message</div>
                            </li>
                            <li class="list-item">
                                <div class="alert alert-info-fl">This is an info alert message</div>
                            </li>
                            <li class="list-item">
                                <div class="alert alert-danger-fl">This is a danger alert messge</div>
                            </li>
                        </ul>

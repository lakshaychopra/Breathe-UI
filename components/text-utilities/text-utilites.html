Text Utilities

Text-Utilities contains various type of text formatting(Sizes, font-style etc)



Headings

Various type of headings can be applied to the according to usage. Either use tags or classes mentioned below to use these component

Heading 1

Heading 2

Heading 3

Heading 4

                    
                        <h1 class="h1">Heading 1</h1>
                        

                        <h2 class="h2">Heading 2</h2>
                        

                        <h3 class="h3">Heading 3</h3>
                        

                        <h4 class="h4">Heading 4</h4>

Font Size

Various type of font-sizes can be applied to the text. Use the classes mentioned below to use these component.

Large Font Size
Medium Font Size
Small Font Size
                    
                        <div class="lg">Large Font Size</div>
                        

                        <div class="md">Medium Font Size</div>
                        

                        <div class="sm">Small Font Size</div>

Text Formatting

Various styles of formatting can be applied to the text using following mentioned classes.

Grey Text
Center Text
                    
                        <div class="grey-txt">Grey Text</div>
                        

                        <div class="center-txt">Center Text</div>

Responsive Images

Responsive images shows visual content on a website in a responsive way. These images are screen responsive.


                    
                        <img class="img-resp" src="path to image" alt="responsive image">

Round Images

To make images round use round-img class with img element.


                    
                        <img class="img-resp" src="path to image" alt="responsive round image">

Avatar

Avatars are images that users can set as their profile picture. They can be custom photos, uploaded by users, or generated as Identicons as a placeholder.



Profile Avatar

Profile Avatar can be created by giving avatar class to the img element

                    
                        <img class="avatar" src="https://url-of-the-image">
                        

                        <img class="avatar" src="path-of-image">

Avatar sizes

Avatars can be given different sizes by giving class names sm | md | lg to the existing avatars

                    
                        <img class="avatar sm" src="https://url-of-the-image">
                        

                        <img class="avatar md" src="path-of-image">
                        

                        <img class="avatar lg" src="path-of-image">

Square Avatar

Avatars can be given square shape by giving class avatar-sq to the img element 

                    
                        <img class="avatar-sq" src="https://url-of-the-image">
                        

                        <img class="avatar-sq" src="path-of-image">

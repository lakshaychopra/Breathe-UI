Button

The Button component is used to trigger an action or event, such as submitting a form, opening a dialog, canceling an action, or performing a delete operation..



Solid Buttons

Solid Buttons are buttons with a filled background

                    
                        <button class="btn btn-primary">Primary</button>
                        

                        <button class="btn btn-secondary">Seconday</button>
                        

                        <button class="btn btn-success">Success</button>
                        

                        <button class="btn btn-danger">Danger</button>
                        

                        <button class="btn btn-warning">Warning</button>
                        

                        <button class="btn btn-info">Info</button>
                        

                        <button class="btn btn-light">Light</button>
                        

                        <button class="btn btn-dark">Dark</button>

Outlined Buttons

Oulined Buttons are buttons without a solid filled background. To use replace btn-variant with btn-outline-variant

                    
                        <button class="btn btn-outline-primary">Primary</button>
                        

                        <button class="btn btn-outline-secondary">Seconday</button>
                        

                        <button class="btn btn-outline-success">Success</button>
                        

                        <button class="btn btn-outline-danger">Danger</button>
                        

                        <button class="btn btn-outline-warning">Warning</button>
                        

                        <button class="btn btn-outline-info">Info</button>
                        

                        <button class="btn btn-outline-light">Light</button>
                        

                        <button class="btn btn-ouline-dark">Dark</button>

Link Buttons

Link buttons are links acting like buttons. Usage is same as other buttons

Primary Seconday Success Danger 
                    
                        <a href="#" class="btn btn-outline-primary">Primary</a>
                        

                        <a href="#"  class="btn btn-outline-secondary">Seconday</a>
                        

                        <a href="#"  class="btn btn-outline-success">Success</a>
                        

                        <a href="#" class="btn btn-danger">Danger</a>

Icon Buttons

To use Icon buttons use class btn-icon with buttons or links

                    
                        <button class="btn btn-icon btn-success">
                            <i class="fa-solid fa-cart-shopping">
                            Add to cart
                        </button>
                        

                        <button class="btn btn-icon btn-primary">
                            <i class="fa-solid fa-thumbs-up">
                            Like
                        </button>

Floating Action Buttons

To use Floating action buttons add class .btn-float to buttons

                    
                        <button class="btn-float btn-info">
                            <i class="fa-solid fa-plus">
                        </button>
                        

                        <button class="btn-float btn-primary">
                            <i class="fa-solid fa-thumbs-up">
                        </button>

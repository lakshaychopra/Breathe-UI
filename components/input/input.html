Input Group

Use the input Group class to the container containing the various inputs

Username

Password

Email

Textarea 
                    
                        <div class="input-group">
                            <span>Username</span>
                            <input type="text" placeholder="Enter Username">
                        </div>
                        

                        <div class="input-group">
                            <span>Password</span>
                            <input type="password" placeholder="Enter Password">
                        </div>
                        

                        <div class="input-group">
                            <span>Email</span>
                            <input type="email" name="" id="" placeholder="Enter your Email">
                        </div>
                        

                        <div class="input-group">
                            <span>Textarea</span>
                            <Textarea placeholder="Enter content"></Textarea>
                        </div>

Input Group with validation

Use the follwing code snippets to add validation over input group.

Password
Enter Password Again
Password does not match

Email
Enter Valid Email
                    
                        <div class="input-group">
                            <span>Password</span>
                            <input class="error" type="password" placeholder="Enter Password">
                        </div>
                        <div class="input-group">
                            <span>Enter Password Again</span>
                            <input class="error" type="password" placeholder="Enter Password">
                            <div class="error-msg">Password does not match</div>
                        </div>
                        

                        <div class="input-group">
                            <span>Email</span>
                            <input class="error" type="email" name="" id="" placeholder="Enter your Email">
                            <div class="error-msg">Enter Valid Email</div>
                        </div>
